# NIM OS INDEX

NIM OS is a decision-aware operating system for the Narrative Intelligence Method.

It separates:

1. Decisions (what is true)
2. State (what is happening now)
3. Execution (how changes are made)
4. Workbench (where analysis happens)

---

## CORE RULES

- All changes must pass through Change Orders
- All decisions must be logged with IDs
- No architectural change without referencing Decision Log
- System evolves, but never silently rewrites history

---

## CURRENT STACK

- Execution: Aider
- Source of Truth: GitHub
- Structuring Engine: AI Studio

---

## SYSTEM MODES

- SYSTEM MODE → repository + backend + funnel logic
- WORKBENCH MODE → narrative analysis + Hearthprint + revision
